# bozhan

BOZHAN framework

dependencies : 
- nodejs 
- redis ( if you want using queue system )
- npm
- typescript


for develop bozhan framework run this commands :

- git clone  https://github.com/mehrabkarimpour-dev/bozhan.git

- cd bozhan

- cp .env.example .env   // in windows => copy .env.example .env

- npm install 

- npm run dev
